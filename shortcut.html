使用「快捷指令」App请先把苹果手机系统升级到 iOS12，并在 App Store 下载「快捷指令」APP 。 捷径社区。
在手机自带Safari浏览器中打开“捷径库”网站，找到所需捷径。
点击“获取捷径”，就可以在“捷径”App中使用新添加的捷径了。

看不到视频？可以新窗口打开试试

1.设置Siri语音播报当日天气

2.设置地理位置
打开捷径-找到文本字样-修改或填写地理位置。
注：手机定位获取地理位置稍微满一些，推荐手写。如果出现慢的状况请更换网络。

3.更换音乐播放器
点击红框（如图）- 更换URL
网易云：orpheuswidget://radio
QQ音乐：qqmusic://today?mid=31&k1=2&k4=0
注：网易云不能换到最近播放，QQ音乐可以。

4.自定义播放组件，如下图所示。
{{gl}} : 公历，如：2019年6月5日
{{week}} : 星期，如：星期二
{{nl}} : 农历，如：己亥年五月初三
{{city}} : 城市，如：北京
{{weather}} : 天气，如：多云
{{winddirect}} : 风向，如：东南风
{{windpower}} : 风力，如：2级
{{temphigh}} : 高温，如：32
{{templow}} : 低温，如：20
{{temp}} : 当前温度，如：23
{{dressIndex}} : 穿衣指数
{{carIndex}} : 洗车指数
{{illnessIndex}} : 感冒指数
{{motionIndex}} : 运动指数
{{zwxIndex}} : 紫外线指数
{{aqi}} : 空气质量，如：优
{{pm2_5}} : PM2.5，如：15
{{aqiInfo}} : 空气质量文案
{{news}} : 新闻
